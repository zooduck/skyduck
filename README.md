# skyduck
Weather forecasting tool for skydivers.

![alt text](https://github.com/zooduck/screenshots/blob/master/skyduck/v0.0.1-alpha/skyduck-v0.0.1-alpha.png)

### APIs
- Dark Sky
- Bing Maps (Geocode)
- Google Maps (Embed)

- dotenv
- full-icu
- express
- express-graphql
- graphql
- axios
- mongodb
